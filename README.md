# TechCorp Website

A simple company website used as a test project for Meeting AI's Coding AI feature.

## Known Issues (intentional bugs for testing)
- Hero title font size is too large (120px)
- Hero title color is red (#ff0000) — doesn't match design
- Subscribe button has no mobile responsive styles
- Subscribe button alert doesn't work properly on mobile

## Structure
```
test-website/
├── index.html
├── src/
│   ├── style.css
│   └── main.js
└── README.md
```

## How to run
Just open `index.html` in a browser.
